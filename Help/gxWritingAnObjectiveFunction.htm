Writing an Objective Function

Objective functions are responsible for providing a measure of "goodness" for each candidate solution proposed by the genetic algorithm.  The performance of each individual must be boiled down to a single real value that gives some indication of how well it solves the problem being optimized.

You can supply an objective function for your problem by attaching an event handler to the CalculateObjective event of the genX GA component.  To do this, first highlight the GA component that you placed on the designer surface and select View | Properties Window to bring up the Properties window.  In this window, select the Event button (the lightning bolt) to display the list of events raised by the component.  Double click in the space after the CalculateObjective event, and you should now see a function body in the main editor window like this:

private void ga1_CalculateObjective(object sender, genX.CalculateObjectiveEventArgs e)
{
  // insert code here, placing objective value in e.Chromosome.RawObjective
}

 

Examining the various samples that ship with genX is the quickest way to learn how to write effective objective functions.
