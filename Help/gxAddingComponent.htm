Adding the genX Component to a Project

This section describes how to add an instance of the genX component to your project.  This section assumes you have already installed genX on your development machine.

Adding a Reference to the genX Assembly  

  • Go to the References node under your project in the Solution Explorer of Visual Studio .NET, and select Project | Add Reference... from the main menu.
  • Select the pane labeled .NET, and click the Browse... button.  In the Open dialog box that opens, navigate to the directory where you installed genX (c:\Program Files\Wanderlust\genX by default), and select the file genX.dll. 
  • Click Open.  You should now see genX.dll listed under the References node in your project.

Adding the genX Component to the Toolbox

  • Select View | Toolbox from the main menu in Visual Studio .NET.
  • Right mouse click inside the Toolbox, and select the Customize Toolbox... menu item from the context menu.
  • Select the pane labeled .NET Framework Components, then click the Browse... button.  In the Open dialog box that appears, navigate to the directory where you installed genX (c:\Program Files\Wanderlust\genX by default), and select the file genX.dll.
  • Click Open.  You should now see a component labeled GA in the Components section of the Toolbox.

Dragging the genX Component to the Design Surface

  • In the Solution Explorer of Visual Studio .NET, select the Windows Form, web page, or other component container to which you want to add the genX component.  You should see the Design View for the container in the main document window.
  • From the Components section of the Toolbox, drag the component labeled GA to the design surface of your container.
  • A component labeled ga1 should appear along the bottom of the design surface. 

 

You can rename the component instance to whatever you'd like, and use the Properties window to view and set properties for the component.  The next section describes the settings and what they do.
