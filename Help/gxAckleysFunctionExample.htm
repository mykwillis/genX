Ackley's Function Example

This example evolves genomes that try to solve a function given by:

F(x) = 20 + e - 20*exp(-0.2*sqrt((1/n)*Sum(i=1,n) {x_i^2})) - exp((1/n)*Sum(i=1,n){cos(2*pi*x_i)})

Encoding

Each genome contains a fixed number of double genes, with each gene specifying the value of one of the parameters x(i) in the candidate solution.

Objective Calculation

The objective is measured as an error function.

Objective = Abs( f(x) )

where x is the vector given by the genes.

Notes

This function has its maximum at {x1, x2, ..., xn} = {0, 0, ..., 0}.
