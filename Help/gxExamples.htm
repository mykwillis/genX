genX Developer's Guide  

Samples

The topics in this section describe the sample programs that ship with genX.  The sample programs can be found in the Samples subdirectory of the directory in which genX was installed.  Each example has its own project file that allows it to be built with Visual Studio .NET.

 

  • Simple Binary.  This example evolves a string of binary genes to a pattern of alternating "0"s and "1"s.
  • Integer Maximization.  This example uses a trivial objective function to evolve chromosomes whose integer genes, when summed, have the highest value.  Fairly useless, but simple enough to prove the thing works.
  • Linear Diophantine Equation .  Just a fancy way of saying that this example tries to evolve solutions {x1, x2, ..., xn} to equations of the form x1y1+ x2y2 + ... + xnyn = 0 where {y1, y2, ..., yn}is given.  Linear meaning to the first power, diophantine meaning having integer value.
  • Single Knapsack.  There's a collection of items with well-defined value and cost.  Choose items such that their combined value is maximized, while keeping their combined cost below some threshold.
