Integer Maximization Example

This example uses a trivial objective function to evolve chromosomes whose integer genes, when summed, have the highest value.  Fairly useless, but simple enough to prove the thing works.

Encoding

Each chromosome contains a fixed number of integer genes.  Each gene is taken directly as an integer.

Objective Calculation

Objective = gene1 + gene2 + ... + genen
