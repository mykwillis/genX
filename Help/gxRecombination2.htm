Combining Genes to Create new Individuals

Supported Recombination Operators

The following recombination operators are supported by genX:

  • Single-point crossover
  • 2-point crossover
  • Uniform crossover
  • Partially Matched Crossover (PMX)

 

The recombination operator to use is specified by setting the genX.GA.Recombinator property to an appropriate delegate.
