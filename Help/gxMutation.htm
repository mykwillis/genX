Introducing New Genes with Mutation

Mutation is a process that alters a gene in an individual, typically by assigning a random allele.  Mutation is applied stochastically with a relatively small probability, on the order of 0.001 mutations per gene copy.

By default, genX uses random mutation to change the value of a gene.  The mutation operator may be selected using the MutationOperator property of the GA component, and the probability can be set with the MutationProbability property.
