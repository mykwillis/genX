Setting Properties at Design Time

Most of the properties of the genX GA component can be set at design time from within the Visual Studio .NET Properties window.  To access these properties, switch to design view for the container of the genX GA component.  Highlight the genX GA component in the design view, and select View | Properties Window (or hit F4) to bring up the Properties window.

The genX properties are split into categories such as General, Encoding, and Recombination.  Within each category, various properties exist that may be set.  Properties that are set in this manner are translated to code by the Visual Studio designer, which is then placed in the InitializeComponent() method of your container.

Highlighting a property in the Properties window shows a brief description of the property at the bottom of the window.  Complete descriptions of all available properties are available in the genX Class Library Reference.
