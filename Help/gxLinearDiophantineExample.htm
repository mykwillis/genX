Linear Diophantine Function Example

This example evolves genomes that try to evolve integer solutions to linear equations.  That is, given an equation of the form:

x1y1+ x2y2 + ... + xnyn = 0

where {y1, y2, ..., yn} is given, the example tries to evolve values of {x1, x2, ..., xn} such that the equation is solved.

Encoding

Each genome contains a fixed number of integer genes, with each gene specifying the value of a coefficient in the candidate solution.

Objective Calculation

The objective is measured as an error function.

Objective = Abs( Sum ( xi * yi ) )
