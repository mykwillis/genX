Single Knapsack Example

The "single knapsack" problem presents the following scenario: you have a knapsack with a fixed capacity C.  There exists a collection of items, each with value vi and required capacity (think of it as cost) ci.  The problem is to find the optimal collection of items to put in your knapsack such that you have the highest total value V, while not exceeding the capacity of the knapsack.

Encoding

A fixed number of binary genes are used, with one gene corresponding to each item.  A false value indicates the item should not be placed in the knapsack, while a true value indicates that it should.

Objective Calculation

The objective calculation adds up the costs of each of the selected items, i.e. those items whose corresponding gene is true.  If the cost exceeds the knapsack capacity C, the objective value is given as a negative value with magnitude equal to the difference between the totaled cost and the capacity.

If the total cost of all selected items is under C, the objective is a positive value equal to the sum of the values of the selected items.
