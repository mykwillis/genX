Simple Binary Example

This example uses a trivial objective function to evolve chromosomes whose binary genes are all 1.

Despite the trivial objective, the example shows the use of the BinaryGene class, as well as demonstrating how to set up event handlers for some of the many events that are fired during execution.

Encoding

Each chromosome contains a fixed number of binary genes. 

Objective Calculation

Objective = gene1 + gene2 + ... + genen
