GA.PropogateParents Property

Gets or sets a value that determines whether or not parents will be propogated to the next generation. 

public bool PropogateParents {get; set;}

Remarks

By default, parents selected for reproduction are not reintroduced into the next generation. Setting this value to true causes the parents, as well as their children, to be added to the next generation.
