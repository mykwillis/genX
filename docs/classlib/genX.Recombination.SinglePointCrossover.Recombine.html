parent1

Return Value

An array of length two containing the offspring from the recombination operation.

Remarks

Single-point crossover generates offspring that have the first half of their genes inherited from one parent and the second half inherited from the other parent. The point at which the switch from one parent's genes to another's is made is called the crossover point.
