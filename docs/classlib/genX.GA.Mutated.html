GA.Mutated Event

Event that is raised when a gene is mutated.

public event MutatedEventHandler Mutated;

Event Data

The event handler receives an argument of type MutatedEventArgs containing data related to this event. The following MutatedEventArgs properties provide information specific to this event.

PropertyDescription
Chromosome The chromosome that was mutated.
MutationPoint The index of the gene at which the mutation occured.
NewGene The value of the gene after the mutation.
OldGene The value of the gene before the mutation.
