GA.MutatePropogatedParents Property

Gets or sets a value that determines whether propogated parents will have a chance of being mutated. 

public bool MutatePropogatedParents {get; set;}

Remarks

This property only has meaning if PropogateParents is true. In this case, this property determines whether or not parents will be subject to the same mutation rate and probability as 'normal' offspring. If it is false, the selected parents will be copied verbatim from one generation to the next.
