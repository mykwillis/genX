genX Genetic Algorithm Library

GA.ValueMutator Property

Gets or sets the function that is called when a gene is to be mutated.

public ValueMutationDelegate ValueMutator {get; set;}

Remarks

Each gene has a default mutation function specified by its associated GeneDescriptor. In most cases, this default mutation function is appropriate.

ValueMutator functions are generally built to handle mutation of a specific type of gene. If the gene population is homogeneous (that is, all genes are of the same type), the ValueMutator property can safely be set to a mutator suitable for that type.

If, however, the gene population is heterogeneous, you must be sure that the ValueMutator specified can deal appropriately with all gene types in the genome.

Set this value to null to use the value mutator specified as the default for a gene.
